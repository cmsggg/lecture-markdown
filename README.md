# \<list\> \<ul\> 문법

### index가 존재하는 list

1. 첫 번째
2. 두 번째
3. 세 번째

### index가 존재하지 않는 list

- 첫 째 원소
  - 첫 째 원소의 첫 째 원소
- 둘 째 원소

- 첫 째 원소
  - 첫 째 원소의 첫 째 원소
  - 첫 째 원소의 둘 째 원소

- 첫 째 원소
- 둘 째 원소
  - 둘 째 원소의 첫 째 원소
