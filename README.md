# \<list\> \<ul\> 문법

### \[ index가 존재하는 list \]

### 방법

```text
1. 첫 번째
2. 두 번째
3. 세 번째
```

### 결과

1. 첫 번째
2. 두 번째
3. 세 번째

### 방법

```text

- 첫 째 원소
- 둘 째 원소

```

### 결과

- 첫 째 원소
  - 첫 째 원소의 첫 째 원소
- 둘 째 원소

- 첫 째 원소

# \<code\> 문법

### 방법

#### 문법 앞 뒤로 어퍼스트로피 3개를 붙여줘야 함
```text 
print("Add Calculator")
a = 10
b = 20
print(a+b)
```

### 결과

```python
print("Add Calculator")
a = 10
b = 20
print(a+b)
```

### 방법

#### 문법 앞 뒤로 어퍼스트로피( \` ) 3개를 붙여줘야 함
```text
<div>
  <h1>title</h1>
  <div>
    <p>contents 1</p>
    <p>contents 2</p>
  </div>
</div>
```

### 결과

```html
<div>
  <h1>title</h1>
  <div>
    <p>contents 1</p>
    <p>contents 2</p>
  </div>
</div>
```
